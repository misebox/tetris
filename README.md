# TETROMINOES

Written in python and implemented using tkinter.

Many bugs have been included now.


## USAGE

```
$ git clone https://github.com/misebox/tetrominoes
$ cd tetrominoes
$ python tetrominoes.py
```

## KEYBOARD MAPPING

|key|description|
|:-|:-|
|ESCAPE|QUIT GAME|
|h, H, ←|MOVE LEFT|
|l, L, →|MOVE RIGHT|
|j, J, ↓|MOVE DOWN|
|R, e, k|ROTATE RIGHT|
|r, E, K|ROTATE LEFT|
|SPACE, ENTER|START GAME / DROP|
